## Usage

```zsh
# installation
$ yarn (or `npm install`)

# development mode
$ yarn dev (or `npm run dev`)

# production mode
$ yarn build (or `npm run build`)
$ yarn start (or `npm start`)
```

And you'll see `Rollup + TypeScript + React = ❤️`!
